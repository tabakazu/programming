## Install
### Java
https://maysatsuki.com/mac/jdk/90

### Kotlin
https://kotlinlang.org/docs/tutorials/command-line.html

```bash
$ brew update
$ brew install kotlin
```

## Compile & Run

```bash
# Compile
$ kotlinc sandbox.kt -include-runtime -d sandbox.jar

# Run
$ java -jar sandbox.jar
```
